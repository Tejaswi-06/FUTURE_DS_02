# 🔄 Customer Retention & Churn Analysis Dashboard

## 📌 Objective
The goal of this project is to analyze customer retention and identify churn patterns to help businesses improve customer satisfaction and reduce customer loss.

## 📁 Dataset Description
The dataset contains customer-related information such as:
- Gender
- Subscription Type
- Contract Type
- Payment Method
- Monthly Charges
- Tenure
- Churn Status

---

## 📈 Key Metrics
- **Total Customers:** 5K  
- **Churned Customers:** 2K  
- **Active Customers:** 3K  
- **Average Tenure:** 36.10 months  
- **Monthly Charges:** 4.30M  

---

## 📊 Key Insights
- Customers with **month-to-month contracts** have the highest churn rate.
- **Fiber internet users** show higher churn compared to DSL.
- **Short tenure customers** are more likely to churn.
- Gender distribution is almost equal, with no major impact on churn.
- Premium subscription users show relatively better retention.

---

## 📉 Visualizations Included
- Customer distribution by gender
- Active customers over years
- Churn trend analysis
- Churn by subscription type
- Contract-wise churn rate
- Internet service impact on churn

---

## 🎯 Conclusion
The analysis highlights that contract type, subscription plan, and service type significantly influence customer churn. Businesses can reduce churn by:
- Promoting long-term contracts
- Improving fiber service quality
- Targeting high-risk customers early
